## 📖 Overview

This project presents a complete financial statement analysis of **Tata Consultancy Services (TCS)** developed entirely in **Microsoft Excel**.

Using publicly available annual reports, historical financial statements were manually extracted, cleaned, and transformed into a dynamic financial model capable of forecasting financial performance, evaluating financial health through key ratios, and presenting insights using an executive dashboard.

The objective was to replicate the workflow of a Financial Analyst while demonstrating practical Excel-based financial modeling skills.

---

## 🎯 Project Objectives

- Analyze TCS's historical financial performance
- Build dynamic linked financial statements
- Forecast future financial performance
- Evaluate profitability, liquidity, efficiency, leverage and growth
- Design an executive dashboard for business reporting
