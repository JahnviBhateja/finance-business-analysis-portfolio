# Dashboards & Models

This section showcases practical finance and business analysis skills through interactive dashboards and financial models.

The projects focus on transforming raw data into meaningful insights using analytical tools.

Typical projects include:

- Financial Performance Dashboard
- Budget vs Actual Analysis
- Revenue & Profitability Dashboard
- Cost Analysis Dashboard
- Financial Ratio Dashboard
- Excel Financial Models
- Power BI Business Dashboards

The objective is not only to visualize data but also to support better business decision-making.
